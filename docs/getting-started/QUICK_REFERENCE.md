# Voiden Keyboard Shortcuts Cheatsheet

Quick reference for all keyboard shortcuts supported in Voiden.

> **Note:** `Cmd` is used on macOS, `Ctrl` on Windows/Linux.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+P` | Open file search |
| `Cmd+Shift+P` | Open command palette |
| `Cmd+Shift+E` | Toggle left sidebar (explorer) |
| `Cmd+J` | Toggle bottom panel (terminal) |
| `Cmd+Y` | Toggle right sidebar (response) |

## Request & Document

| Shortcut | Action |
|----------|--------|
| `Cmd+Enter` | Send HTTP request |
| `Cmd+S` | Save document |

## Find & Replace

| Shortcut | Action |
|----------|--------|
| `Cmd+F` | Find in document |
| `Cmd+H` | Find and replace |
| `Cmd+G` | Find next match |
| `Cmd+Shift+G` | Find previous match |
| `Enter` | Find next (in find field) |
| `Shift+Enter` | Find previous (in find field) |
| `Cmd+Enter` | Replace all (in replace field) |
| `Esc` | Close find/replace panel |

## Block Editing

> These shortcuts only work when the block context menu is open (`Cmd+.`).

| Shortcut | Action |
|----------|--------|
| `Cmd+.` | Open block menu |
| `Cmd+D` | Duplicate block |
| `Cmd+C` | Copy block |
| `Cmd+X` | Cut block |
| `Cmd+L` | Link block |
| `Cmd+Shift+Up` | Add block above |
| `Cmd+Shift+Down` | Add block below |
| `Backspace` | Delete block |

## Command Palette

| Shortcut | Action |
|----------|--------|
| `Arrow Up/Down` | Navigate items |
| `Enter` | Execute command |
| `Tab` | Autocomplete path |
| `Esc` | Close palette |

## Code Editor

| Shortcut | Action |
|----------|--------|
| `Cmd+/` | Toggle comment |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+[` | Decrease indent |
| `Cmd+]` | Increase indent |
| `Ctrl+Space` | Trigger autocomplete |
| `Tab` | Accept autocomplete |
| `Esc` | Close autocomplete |

## Terminal

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Copy selected text |
| `Ctrl+V` | Paste |
| `Ctrl+A` | Select all |
| `Ctrl+L` | Clear terminal |

## File Explorer

> These shortcuts only work when the file context menu is open (right-click).

| Shortcut | Action |
|----------|--------|
| `Option+Cmd+R` | Reveal in Finder |
| `Cmd+Backspace` | Delete file/folder |
